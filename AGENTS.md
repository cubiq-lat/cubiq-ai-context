## Git Commits & Pushes

- Confirm what will be committed first
- If a `package.json` exists, update the version in it. By default, increment the patch version (last number) by 1. Only update minor or major version when explicitly instructed.
- Commit with message starting with "AGENT:" prefix
- Never push automatically — always ask first
